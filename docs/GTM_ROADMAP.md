# Qualy GTM Roadmap

*Last updated: 2026-05-12*

> Goal: start showing Qualy to the market before every asset feels perfect. This roadmap favors small, repeatable publishing loops over a big launch package.

## Positioning Decisions

- [x] Position Qualy as a multi-channel AI messaging assistant, not a WhatsApp-only product.
- [x] Use the supported channel set consistently: WhatsApp, Instagram, Telegram, and Facebook Messenger.
- [x] Show all four channel logos when channel coverage is part of the visual.
- [x] Vary chat mockups by asset: some visuals can use WhatsApp-style conversations, others Instagram DM-style conversations.
- [x] Keep Turkish market as the primary GTM target.
- [x] Use Turkish-first visual copy for Instagram and Turkish SMB outreach.
- [x] Use vertical 4:5 as the default Instagram feed visual format.
- [x] Omit the Qualy logo from normal AI-generated social concepts unless explicitly requested.
- [x] Keep visual copy short and stop-the-scroll: one punchy slogan plus at most one support line.
- [x] Create English variants only for Product Hunt and selected LinkedIn posts.
- [x] Keep captions Turkish-first unless the target audience is explicitly international.

## Core GTM Principle

- [x] Treat the first GTM phase as a learning loop, not a perfect brand campaign.
- [x] Prioritize publishing consistency over custom design work for every post.
- [x] Reuse one approved visual system across Instagram, LinkedIn, and Product Hunt.
- [ ] Keep every asset grounded in realistic demo conversations, not invented customer claims.
- [ ] Avoid unsupported metrics, fake testimonials, and fake customer logos.
